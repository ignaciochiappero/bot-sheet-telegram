# Curso: Cómo funciona el Bot de Telegram + Google Sheets

## Overview

Este es un curso en olas que explica cómo está construido el bot. Cada ola agrega una capa de complejidad sobre la anterior.

**Orden sugerido:**

1. **[OLA 1: Panorama General](./01-ola-1-panorama-general.md)** — Las 4 piezas del sistema
2. **[OLA 2: Telegram + Webhooks](./02-ola-2-telegram-y-webhooks.md)** — Cómo llega el mensaje
3. **[OLA 2B: API Gateway](./02b-ola-2b-api-gateway.md)** — El portero entre internet y Lambda
4. **[OLA 3: AWS Lambda](./03-ola-3-aws-lambda.md)** — Compute serverless
5. **[OLA 4: grammY](./04-ola-4-grammy.md)** — El framework del bot
6. **[OLA 5: Groq AI](./05-ola-5-groq-ai.md)** — La inteligencia artificial
7. **[OLA 6: Google Sheets](./06-ola-6-google-sheets.md)** — La base de datos
8. **[OLA 7: Terraform](./07-ola-7-terraform.md)** — La infraestructura

---

## Cómo usar este curso

### Si sos principiante

Empezá por la OLA 1 y avanzá en orden. Cada ola assume que entendiste las anteriores.

### Si ya sabés algo

Podés saltar a la ola que te interese. Cada una es autocontenida.

### Para referencia rápida

Las olas están diseñadas para reread después de haberlas leído una vez.

## Requisitos

- Ganas de aprender
- Nociones básicas de JavaScript/TypeScript (opcional, las olas explican sin código también)
- 30 minutos por ola (leyendo con calma)

- **Referencia**: Podés volver a una ola específica después
